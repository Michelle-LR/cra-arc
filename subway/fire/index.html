File electronically: Information returns (tax slips and tax summaries)

When and how to file, distribute and amend information slips electronically (for example, T4, T4A, T5, and other type of returns), including how to use Web Forms.

Filing electronically lets you:

  • Get confirmation within minutes
  • File securely
  • Make changes
  • Allow another authirized source, such as a tax preparer, to file for you
  • Send returns for the previous ten years
