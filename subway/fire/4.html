What you need to file your information return electronically

Before you file, have your documentation ready. You will also need your account number (Trust Number, Non-Resident Number, 15 characters RP or RZ Business Number) and Web access code (WAC).

What is a Web access code (WAC)?

A Web access code is a personalized code that is associated with your account number (RP, RZ, Trust or Non-Resident). Your Web access code contains two letters and four numbers. It is case sensitive.

Where do you find your Web access code (WAC)?

If you do not remember your code or you need a new one, you can:

  • use our Web access code online service
  • call business enquiries at 1-800-959-5525

What if you have a balance due?

When you make a payment to the Canada Revenue Agency, do not resubmit your return.

What are the filing methods for electronic

Where possible, you should file returns electronically to avoid the delays that may occur in processing paper returns.
Note: We do not accept returns filed on CDs, DVDs, USB keys.

Web Forms: Option 1

Web Forms allows you to complete an information return easily, up to 100 slips, following step-by-step instructions. To use it, an account number and the Web access code (WAC) associated to this number is required.

  • Benefits to using Web Forms
  • Completely FREE
  • You can file up to 100 slips (original, amended, or cancelled) from our website. If you want to file additional slips, file them as original slips
  • Create an electronic information return containing slips and a summary, which can be saved and imported at a later date
  • Summary totals are calculated for you
  • Allows you to print your slips and your summary
  • The application validates data in real time
  • Send encrypted returns over the Internet

OR

Internet file transfer: Option 2

Internet file transfer allows you to use payroll, commercial, or in-house developed software, to submit an XML file of up to 150 MB over the Internet. To use it, an account number and the Web access code (WAC) associated to this number is required.
IMPORTANT : An .xml file created with Web Forms or a PDF document saved as an .xml will NOT pass validations through Internet file transfer.

  • Details about XML
  • If you use commercial software, it will automatically create your .xml file. Use Internet file transfer to submit your file.
  • If you do not use commercial software, the use of the Web Forms is recommended, otherwise, you have to create your own .xml file.
  • If you are a software developer and you want to create your own software, you need the XML specifications to file an information return. Got to Find XML specifications to file electronically for more detail.
  • Download the CRA schema; to file information returns in 2023, download the xmlschm1-23-1 file. (version 1-23-1, last updated October 7, 2022)

After you save the .zip file on your computer, use an extracting software (for example WinZip) to extract the .xsd files. You have to extract and save all files to one location on your hard drive.

  • Validate your file using a validating parser downloaded from the Internet.

Other methods of electronic filing:

My Business Account is a secure, password protected portal, that allows you to file information returns electronically through Internet file transfer and Web forms.
Represent a Client is a secure service to access tax information and file information returns electronically on behalf of individuals and businesses, including your employer.

What happens if you have more than one payroll account

If you have more than one payroll account, you have to file a separate return for each account.

What happens if you file using the wrong method

If you are required to file electronically and you do not file the returns by Internet file transfer or Web Forms, you may have to pay a penalty as determined in the table below.
Each slip is an information return, and the penalty the CRA assesses is based on the number of information returns filed in an incorrect format. The penalty is calculated per type of information return. For example, if you file 51 T5 slips and 51 T4 slips on paper, the CRA would assess 2 penalties of $250, 1 per type of information return.

What if you file your information return late?

If you file your information return past the due date you may be charged a penalty. To learn more, see late-filing penalties for information returns.
