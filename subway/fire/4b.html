Filing with Internet file transfer

Internet file transfer allows you to use payroll, commercial, or in-house developed software, to submit an XML file of up to 150 MB over the Internet.

Steps

  1. Have your XML file ready

    Have your XML file completed and save as .xml.

    Important considerations:

    • If you are using a commercial software, it will automatically create your .xml file
    • IMPORTANT An .xml file created with Web Forms or a PDF document saved as an .xml will not pass validations through Internet filer transfer

  2. Have your account number and Web access code (WAC) associated to this number

    To use the Internet file transfer (XML) application, your valid account number and the Web access code (WAC) associated to this number is required.

    You can also have access Internet file transfer (XML) through My Business Account and Represent a Client.

  3. Log in the Internet file transfer (XML) application

    You can now browse and load your .xml file in the Internet file transfer (XML) application to send your declaration to CRA.
