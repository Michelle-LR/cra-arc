Customers

If you are only buying products or services using digital platforms (not selling or facilitating sales), the new requirements do not apply to you. You do not have to share, report, or file anything. Your browsing and buying information is also not shared with the CRA.

Sellers of products or services

If you are a seller of products or services who is considered to be a reportable seller, your role under the rules is done once you have shared the reportable elements with your platform operators.

However, you may still have other tax obligations with the CRA and need to report your earnings. For more information, refer to: the CRA’s Taxes and the platform economy

Digital platform operators

If you are a digital platform operator, and considered to be a reporting platform operator, you must obtain a “RZ” business number account prior to filing a Part XX Information Return for the first time.

The way in which you file will be different based on if you are a resident in Canada or not:

  • Platforms resident in Canada will be able to file through My Business Account.
  • Platforms not resident in Canada you will need to first obtain a Web Access Code in order to electronically file your return through the CRA web portal.

The Part XX Information Return must be filed with the CRA in prescribed form using an XML Schema.

XML Schema – Internet file transfer (XML) allows you to transmit original, amended or fixed returns with a maximum file size of 150 MB. All you need is a web browser to connect to the internet, and your software will create, print, and save your electronic information returns in XML format.
