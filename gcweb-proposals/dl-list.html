Description list (horizontal) extension

Finding ways to extend the GCWeb theme to <dl class="dl-horizontal"> lists which were develped for WET 4 and a smaller font size.

Background

The horizontal description list design was developed for the WET4 theme and has been a great design. However, the GCWeb theme has increased the WET4 font size from 16px, to 20px. In turn, the width of the <dt> in horizontal mode has not increased in width, thus providing an uneven experience between the two themes. More words are at risk of being truncated (especially in French).

In the examples below, the second term item would not be truncated, or wrapped in WET4, but it does in GCWeb theme.
